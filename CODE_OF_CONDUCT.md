# Uppförandekod

Vi vill att Open Data Hub SE ska vara en välkomnande, vänlig och trygg plats för alla.

- Var hjälpsam och respektfull mot andra
- Ingen diskriminering, trakasserier eller personangrepp
- Ge konstruktiv feedback
- Tänk på tonen – vi är här för att lära och dela

Om du ser olämpligt beteende, kontakta en moderator eller öppna ett ärende.

Tillsammans bygger vi ett öppet och inkluderande community!
