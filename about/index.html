Om Open Data Hub SE

Vad är Open Data Hub SE?

Open Data Hub SE är en community-driven plattform för att lära sig om, arbeta med och dela öppna data i Sverige. Vi tror på kraften i öppen data för att skapa transparens, driva innovation och lösa verkliga problem.

Vår Mission

Vi vill göra det enkelt för utvecklare, datakonsulter, journalister och medborgare att:

  • Hitta och förstå öppna datakällor i Sverige
  • Lära sig bästa praxis för att arbeta med data
  • Bygga meningsfulla projekt och analyser
  • Samarbeta och dela kunskap med andra

Vem kan delta?

Alla är välkomna! Oavsett om du är erfaren datatekniker, student eller bara nyfiken på data - det finns något för dig här.

Hur börjar jag?

Börja med att läsa våra guider, utforska resurser, eller gå med i communityn.
